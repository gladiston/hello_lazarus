Instalação de Pacotes via OPM: HtmlViewer

THtmlPort é uma suíte de componentes(THtmlViewer, TFrameViewer e TFrameBrowser) que nos permitem exibir texto formatado em HTML.

Não é um navegador de internet, mas considere que você pode exibir um código HTML puro com hyperlink e tudo mais, código HTML que pode ser exibido através dum index.html salvo no disco ou um texto HTML que enviará dinamicamente ao componente.. A maneira mais simplista é como dizer que THtmlViewer é como TLabel que pode lhe mostrar um código HTML formatado.

Considero ele essencial em minhas aplicações. Dá para utilizá-lo também no Delphi na versão comercial e Comunity Edition.

Leia a documentação no link para saber como utilizá-lo.
