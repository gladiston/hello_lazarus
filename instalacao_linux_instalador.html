Instalação no Linux: Instalação a partir dos repositórios

A instalação a partir dos repositórios é recomendada apenas ao FreePascal(FPC), porque o Lazarus precisa de uma instalação será homeuser, caso contrário você teria de dar permissões a arquivos que o instalador do pacotes agregar e isso complica muito desnecessariamente.

Nesta instalação, instalaremos o usaremos o fpcupdeluge para instalar o FPC dos repositórios, porém o Lazarus no HOME do usuário. Primeiro vamos instalar alguns pacotes essenciais:

     sudo dnf install make automake gcc gcc-c++ kernel-devel
     sudo dnf groupinstall "Development Tools" "Development Libraries"
     sudo dnf install gdb zip unzip gdb subversion binutils libX11-devel gtk2-devel gdk-pixbuf2-xlib
     sudo dnf install cairo cairo-devel cairomm-devel 
     sudo dnf install pango pango-devel pangomm pangomm-devel

Se planeja compilar o FPC separadamente, então inclua:

     sudo dnf install fpc fpc-src qt5pas-devel

Com o FPC instalado no seu sistema você está apto a instalar o Lazarus via GIT ou fpcupdeluge. Para não ser muito repetitivo, escolha como desejará fazer a instalação do Lazarus:

  2. Instalação no Linux: Usando GIT
