Instalação de Pacotes opcional via OPM: CEF4Delphi

Use o motor do chromium no Delphi/Lazarus. Requer instalação de runtime (conjunto de DLLs) externo para funcionar. Também é possível instalá-la a partir do repositório oficial:

Repositório Oficial

Dá para instalar a partir dos repositórios do Lazarus, contudo o runtime deve ter a mesma versão do pacote que foi instalado, assim não adianta baixar o runtime do git mais atualizado(ex. 90.5.9.0) se você baixou e instalou o CEF na versão 90.5.4.0, pois não irá funcionar. Por essa razão, é melhor baixar a partir do repositório oficial, pois o runtime estará disponível na pasta de instalação com a mesma versão do runtime a ser distribuído junto com o executável.
