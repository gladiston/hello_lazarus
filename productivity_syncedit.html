Produtividade: Refatorando nomes numa seleção de linhas

As vezes num bloco de texto marcada desejamos alterar o nome de variaveis ou objetos, claro que sempre poderemos usar o search/replace. Porém há uma alternativa melhor, renomear enquanto digita e trocar todas as ocorrências dentro da seleção. Como é mais dificil explicar do que mostrar, então veja o vídeo e aprenda:

Use este recurso sempre que desejar renomear um objeto dentro de um bloco selecionado, nunca use este recurso para uma unit inteira, para este caso você deve usar os recursos de Refactoring que a IDE dispõe.
