Instalação de Pacotes: Introdução ao complementos, instalar os básicos

Os pacotes estendem as funcionalidades da IDE e a programação, eles podem conter classes de objetos e/ou componentes visuais/não visuais. O Lazarus vem com muitos deles, mas boa parte não está instalada, são complementos adicionais que você decide instalar ou não pós-instalação. É uma boa recomendação instalar ou concentrar-se nos pacotes que precisa ter como base para os projetos, evite adicionar o que não ou nem sabe para que serve.

Abaixo eu mostro a lista de complementos básicos que costumo instalar:

  1. imagesforlazarus:Permite que a IDE e seus programas possam lidar com vários formatos de imagens como o jpeg. Leia o README.TXT na pasta \components\images
  2. lazdbexport:Ainda não testei, mas fornece a alguns componentes(TDataset) a possibilidade de exportar seus dados para alguns formatos externos como csv.dbf,rtf,...
    Página explicativa
  3. AnchorDocking:Explicada neste artigo
  4. AnchorDockingDsgn:Explicada neste artigo
  5. dockedformeditor:Explicada neste artigo

Depois posso até instalar outros ou não, mas estes daí são instalados assim que a IDE completou a instalação.
