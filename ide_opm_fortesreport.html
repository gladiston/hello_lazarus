Instalação de Gerador de relatórios: Fortes Report

Há duas grandes opções de relatórios: o LazReport e o Fortes-CE, neste passo vamos instalar o Fortes-CE.

´

Geralmente tem muitos HowTo´s no YoutTube demonstrando sua utilização. Este pacote é instalável também no Delphi versão comercial e Comunity Edition. Criar relatórios com ele requer programação por bandas, similar ao antigo Quickreport. Programação por bandas é bem conveniente porque você usar o próprio formulário para escrever o relatório que por sua vez terá componentes visuais apropriados para ele e componentes não visuais de qualquer espécie que possa te ajudar a preparar o relatório. Com eventos como OnBefore/OnAfter eles permitem usar o código pascal para mutacionar/ocultar a apresentação das informações.


IMPORTANTE #1: Não misture a instalação desse pacote com outra simultânea, instale apenas e ele, e depois se precisar instale os demais pacotes.

IMPORTANTE #2: Muito cuidado para não instalar também o Fortes4Lazarus, trata-se do mesmo componente, mas ele foi renomeado para “Fortes-CE”. Mantiveram o nome antigo para não quebrar as dependências com códigos mais antigos.
