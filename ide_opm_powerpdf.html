Instalação de Pacotes via OPM: PowerPDF

PowerPDF é um conjunto de componentes não visuais com o qual podemos criar documentos PDF.

Os documentos em PDF gerados são no formato PDF 1.2. Cuidado, este componente serve para criar PDF e não para lê-los. Talvez ache que nunca a usará, mas o fato é que alguns geradores de relatórios a utilizam, então recomendo que faça a instalação.

Maiores informações em:
https://wiki.lazarus.freepascal.org/PowerPDF
